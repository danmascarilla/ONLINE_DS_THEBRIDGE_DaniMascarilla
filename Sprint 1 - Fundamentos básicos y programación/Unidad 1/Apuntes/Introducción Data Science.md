*¿Que es?

Es una disciplina que se encarga de extraer conocimiento y patrones para tomar decisiones a partir de ellos

Datos, materia prima

Lenguaje para la ciencia de datos es el *Python*

*Roles y funciones

- Recolección y procesamiento de datos
- Análisis exploratiorio
- Modelado y machine learning
- Comunicación de resultados
- Optimización

*Donde se aplica la ciencia de datos (ejemplos)

- Análisis de datos y extracción de info (meteorologia. enfermedades, etcétera)
- Algoritmos predictivos y prescriptivos (valor futuro de la bolsa, mejor ruta para llegar a un destino)
- Automatización de procesos (aprendizaje de tareas por parte de robots, ...)

Se aplica para dar sentido a la información a una empres, también para imagenes para dar sentido a enfermedades con el deep lerning, también en la bolsa con los algoritmos predictivos, con la IA asistentes virtuales y chatbots
