 **¿Por qué Git y GitHub?**

1. Son los más extendidos y soportados
    
2. Los vas a utilizar en tu vida profesional para desarrollar en equipo
    
3. Los vamos a emplear en el curso.

Git es un sistema de control de versiones de código que permite guardar y mantener un proyecto desarrollado por una o varias personas de forma eficiente.

- Repeticiones innecesarias altamente redudantes
- Gestión de trabajo concurrente en un mismo proyecto

Repositorios
- Lugar donde se almacenan los archivos de un proyecto bajo control de versiones. Puede ser local, en el ordenador del usuario o remoto, en servidor, web.
