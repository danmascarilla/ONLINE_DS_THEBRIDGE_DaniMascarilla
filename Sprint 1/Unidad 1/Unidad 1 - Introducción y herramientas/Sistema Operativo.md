	Comandos del sistema Operativo

*Powershell*

Abrir consola --> Windows + R "cmd" + "powershell"

Donde estamos --> pwd
Crear directorio --> mkdir carpeta
Crear subdirectorio --> mkdir carpeta\carpeta2
Moverte entre directorios --> cd
Saber que archivos hay en x directorio --> dir
Renombrar carpetas --> ren carpeta proyectos
Crear archivo vacio --> echo "" > test.txt
	 echo "Hola mundo" > test.txt
Borrar archivo --> rm test.txt
